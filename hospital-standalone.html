歩行再建への提案

未入力

確率を計算すると、歩行再建への提案が表示されます。

計算式

score = -10 x Age65 + 2 x L3 Motor + 2 x S1 Motor + 5 x L3 LT + 5 x S1 LT P = exp(-3.273 + 0.267 x score) / (1 + exp(-3.273 + 0.267 x score))

本ツールはvan Middendorpらの臨床予測ルールに基づく補助ツールです。予測対象は1年後の屋内歩行自立であり、屋外歩行能力、歩行速度、歩行の質を直接予測するものではありません。臨床判断を置き換えるものではありません。

計算式・出典

予測ルール

Prediction score = -10 x Age65 + 2 x L3 Motor + 2 x S1 Motor + 5 x L3 Light touch + 5 x S1 Light touch

P = exp(-3.273 + 0.267 x score) / (1 + exp(-3.273 + 0.267 x score))

出典: van Middendorp JJ, Hosman AJF, Donders ART, et al. A clinical prediction rule for ambulation outcomes after traumatic spinal cord injury: a longitudinal cohort study. Lancet. 2011;377:1004-1010.
